# Структура CSV-выгрузки

Если у собственника не подключена бухгалтерия (`~~бухгалтерия`) или платежи (`~~платежи`), он может выгрузить CSV/Excel вручную (например, из банк-клиента, ЮKassa, 1С). Этот документ описывает, какие колонки ожидать и как обрабатывать варианты.

## CSV выручки (выгрузка операций)

Ожидаемые колонки (порядок неважен, регистр заголовков неважен):

| Колонка | Обязательна | Описание |
|---|---|---|
| `дата` / `date` | Да | Дата операции (любой стандартный формат: ДД.ММ.ГГГГ, ГГГГ-ММ-ДД и т.д.) |
| `товар` / `услуга` / `позиция` / `назначение` | Да | Что продано |
| `сумма` / `выручка` / `итого` / `amount` | Да | Сумма операции (₽) |
| `количество` / `кол-во` / `qty` | Нет | Продано единиц — если нет, считаем 1 на операцию |
| `ндс` / `ставка_ндс` | Нет | Ставка НДС по позиции (без НДС / 5% / 7% / 22%) — если есть, помогает очистить выручку |

**Выгрузки, которые обычно подходят:**
- Банк-клиент: выписка по счёту → экспорт в CSV/Excel
- ЮKassa / Тинькофф Касса: раздел операций → выгрузка
- 1С / МойСклад: отчёт по продажам → экспорт

Если в выгрузке больше колонок — лишние игнорируйте. Если обязательной колонки нет — спросите собственника, какая колонка ей соответствует.

## CSV затрат (выгрузка расходов или себестоимости)

Ожидаемые колонки:

| Колонка | Обязательна | Описание |
|---|---|---|
| `дата` / `date` | Нет | Дата расхода (полезна для анализа тренда) |
| `товар` / `услуга` / `категория` | Да | К чему относится затрата |
| `сумма` / `затрата` / `расход` / `cost` | Да | Сумма затраты (₽) |
| `тип` / `type` | Нет | Себестоимость или операционный расход — если нет, спросите собственника |

**Выгрузки, которые обычно подходят:**
- 1С:Бухгалтерия / Контур.Эльба: ОПиУ (детально) → экспорт
- Банк-клиент: операции по расходам → фильтр по категории → экспорт

## Обработка «грязных» CSV

Реальные выгрузки беспорядочны. Частые проблемы:

- **Лишние строки-шапки:** пропускайте строки, пока не найдёте похожую на названия колонок
- **Символы валюты:** убирайте `₽`, `$`, пробелы-разделители тысяч и приводите запятую-десятичную к точке перед разбором числа
- **Возвраты с минусом:** включайте их — они уменьшают чистую выручку
- **Смешанные валюты:** отметьте и спросите, какую брать; по умолчанию рубли
- **«Брутто» против «нетто»:** для выручки предпочтительно нетто (после комиссии эквайринга); если неясно — спросите
- **НДС в суммах:** если в выгрузке нет колонки НДС, а собственник — плательщик, спросите ставку и очистите выручку (сумма ÷ (1 + ставка))

## После загрузки

Подтвердите форму данных с собственником, прежде чем продолжать:
- «Загрузил X операций с [дата] по [дата] по Y позициям. Всё верно?»
- Если период или число позиций выглядит странно — попросите перепроверить фильтры выгрузки.
