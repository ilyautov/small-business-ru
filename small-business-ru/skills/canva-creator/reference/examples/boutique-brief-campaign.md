# Разобранный пример: бриф бутика → кампания

**Бизнес:** «Лён и Нить» — женский бутик одежды, Краснодар
**Тариф `~~дизайна`:** командный (VistaCreate, без API брендовых шаблонов)
**`~~crm`:** Битрикс24 с модулем CRM-маркетинга (планирование публикаций)
**Источник брифа:** вывод скилла content-strategy (согласован владельцем)

---

## Вход: согласованный бриф (резюме)

```
Продвигать активно: льняные летние платья (топ-продавец, рост +22% м/м)
Держать ровно: аксессуары
Сезонная возможность: переход к школьному сезону (сдвиг 15.08)
Акция: при покупке 2 платьев — аксессуар в подарок (с 01.06 по 31.07)
Каналы: лента Telegram, VK, сторис VK
Частота: 3× в неделю, пн/ср/пт, 10:00 МСК
Голос: тёплый, разговорный, лёгкий юмор. Без корпоративщины.
```

---

|------|-------|------|------------|--------------|
| 02.06 | Лента Telegram | Запуск льняных платьев | Квадратный пост | Знакомство — «наконец-то платье, в котором можно и туда, и туда» |
| 04.06 | Лента VK | Льняные платья | Горизонтальный пост | Соцдоказательство — «самая разбираемая вещь этой весны» |
| 06.06 | Сторис VK | Акция «2+1» | Сторис (вертикаль) | Срочность акции — переход в магазин |
| 09.06 | Лента Telegram | Совет по стилю | Квадратный пост | Польза — «3 способа носить льняное миди» |
| 11.06 | Лента VK | За кулисами | Горизонтальный пост | История бренда — как их шьют |
| 13.06 | Сторис VK | Образ на выходные | Сторис (вертикаль) | Лайфстайл — «твоя суббота — собрана» |
| … | … | … | … | … |

*Владелец согласовал — «перенеси 11.06 на 12.06, 11-го я пощу про распродажу. В остальном отлично».*

## Стадия 2: генерация макетов (базовый/командный путь без API-шаблонов)

**Обмен по выбору шаблона:**

```
Claude: Нашёл 3 шаблона квадратных постов в вашей библиотеке VistaCreate:
  1. Бутик Квадрат — Кремовый  (правлен апр. 2026)
  2. Летняя акция — Песочный    (правлен март 2026)
  3. Карточка товара — Жирный   (правлен фев. 2026)
  Какой использовать для постов в ленту?

Владелец: Кремовый, №1.

Claude: Принято. Использую «Бутик Квадрат — Кремовый» для всех постов в ленту.
        Создаю заготовки дизайнов — дам ссылки, чтобы вы вставили фото товаров
        и текст в VistaCreate, потом вернётесь, и я экспортирую.
```

*[Дизайны созданы; владелец заполняет в VistaCreate; возвращает ID дизайнов]*

**Вывод экспорта:**
```
Экспортирую превью...

✓ Пост VK 04.06          → https://<сервис>/export/def456.png
✓ Сторис 06.06           → https://<сервис>/export/ghi789.png
[…]
```

*Владелец: «Выглядит отлично. Давай дальше.»*

## Стадия 3: черновики подписей

**02.06 — лента Telegram (запуск льняных платьев)**
```
Наконец-то платье, в котором можно и на рынок, и на ужин. 🌿

Наше льняное миди только приехало и уже самая запрашиваемая вещь этого лета.
Дышит, сидит легко, и да — у него есть карманы.

Забирайте, пока размеры на месте → ссылка в описании.

#ЛьняноеПлатье #КраснодарСтиль #ЛетняяМода #ЛёнИНить
```

**04.06 — лента VK (соцдоказательство)**
```
«Надела четыре раза за неделю и ни капли не жалею.»

Это реальный отзыв реальной покупательницы про наше льняное миди. Между нами —
мы не удивлены. Смотрите коллекцию по ссылке ниже.
```

**06.06 — сторис VK (акция)**
```
👗 Купи 2 платья → аксессуар В ПОДАРОК

Акция с 01.06 по 31.07. Не проспите.

Переходите в магазин →
```

*Владелец согласовал с одной правкой: «В подпись 04.06 добавь цену в конце. "7 200 ₽, ссылка ниже".»*
*Подпись обновлена.*

## Стадия 4: постановка в очередь `~~crm`

**Кампания создана:**
```
Кампания: «Лён и Нить — Летний лён 2026»
Старт: 02.06.2026 | Финиш: 31.07.2026
ID кампании: 8a72c1f4...
```

**Очередь публикаций (первые 3):**

| Запланировано | Канал | Превью подписи | Статус |
|---------------|-------|----------------|--------|
| 02.06, 10:00 МСК | Telegram | «Наконец-то платье, в котором можно…» | ЗАПЛАНИРОВАНО |
| 04.06, 10:00 МСК | VK | «"Надела четыре раза за неделю…"» | ЗАПЛАНИРОВАНО |
| 06.06, 10:00 МСК | Сторис VK | «Купи 2 платья → аксессуар В ПОДАРОК…» | ЗАПЛАНИРОВАНО |

**Финальное сообщение владельцу:**
```
Ваша кампания поставлена в очередь в Битрикс24:
https://<ваш-портал>.bitrix24.ru/marketing/campaigns/8a72c1f4

Посты опубликуются автоматически по расписанию. Любой пост можно отменить или
отредактировать в CRM до выхода. Что поправить, прежде чем закончим?
```

> **Маркировка рекламы:** все эти посты — органические, в собственных каналах
> бренда, маркировки не требуют. Если бы акцию «2+1» продвигали **платно**
> (Яндекс Директ, VK Реклама, посевы в чужих каналах) — креативы потребовали бы
> токена ОРД и передачи данных в ЕРИР. ⚠️

## Пример письма (только текст — без `~~дизайна`)

Email-строки полностью пропускают дизайн-сервис. Claude пишет тему, прехедер и тело письма обычной прозой на Стадии 3, а владелец копирует контент в свой сервис рассылки (`~~почта`, Unisender, Sendsay), когда готов отправить.

**Строка календаря:**

| Дата | Канал | Тема | Тип ассета | Угол |
|------|-------|------|------------|------|
| 05.06 | Email | Запуск летнего льна | Письмо (текст) | Главная выгода клиенту + акция |

**Вывод Стадии 3 (инлайн, без дизайна):**

```
Тема: Лён, который правда дышит
Прехедер: Новые летние модели — и небольшой подарок, если возьмёте две.

Первая партия льняных миди только поступила. Это то платье, в котором можно
пойти на бранч, на свадьбу или забрать ребёнка из школы — и не чувствовать,
что делал выбор гардероба.

Что стоит знать:
  • Размеры XS–XL, четыре цвета
  • Карманы (настоящие)
  • Купите 2 платья до 31 июля — добавим аксессуар в подарок

Заходите на ул. Красную, 41 в эти выходные или смотрите онлайн по ссылке ниже.

Смотреть льняную коллекцию →

— Майя
Лён и Нить
```

**Передача на Стадии 5:**

```
Ваши посты для соцсетей поставлены в очередь в Битрикс24: [ссылка]

Письмо на 5 июня готово выше — скопируйте его в свой сервис рассылки
(Unisender, Sendsay), когда будете готовы отправить. Письма мы напрямую в
очередь не ставим — так вы сначала проверите вёрстку в своей платформе отправки.
```
