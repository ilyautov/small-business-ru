# Справочник по формату пакета закрытия

Пакет закрытия — два файла: xlsx-книга и PDF-сводка на одну страницу.

## Именование файлов

```
close-packet-2026-03.xlsx
close-packet-2026-03-summary.pdf
```

Используй год-месяц в формате ISO 8601 (`ГГГГ-ММ`) в имени файла. Расположение
по умолчанию — Рабочий стол; используй предпочитаемый путь собственника, если
он указал.

---

## xlsx-книга — три листа

### Лист 1: ОПиУ

Отформатированная копия ОПиУ из бухгалтерии за закрываемый месяц. Раскладка в
два столбца: **Статья** и **Сумма**.

Обязательные строки (по порядку):
1. Итого выручка
2. Итого себестоимость
3. **Валовая прибыль** (жирным)
4. **Валовая маржа %** (жирным, в формате %)
5. Операционные расходы по статьям (каждая на своей строке)
6. Итого операционные расходы
7. **Чистая прибыль** (жирным)

Включи столбец сравнения м/м (месяц к месяцу), если данные прошлого месяца
доступны. Форматируй суммы как валюту (`# ##0,00 ₽`). Отрицательные значения —
красным.

### Лист 2: Сверка

Сопоставление поступлений по банку в учёте с зачислениями эквайринга, рядом.

Столбцы:
| Столбец | Источник |
|---|---|
| Дата (учёт) | Дата поступления в учёте |
| Сумма (учёт) | Сумма поступления в учёте |
| Источник | ЮKassa / Тинькофф / банк / СБП |
| Дата (зачисление) | Дата зачисления по эквайрингу/банку |
| Сумма (зачисление) | Сумма зачисления (за вычетом комиссии) |
| Дельта | Сумма в учёте минус сумма зачисления |
| Статус | СВЕРЕНО / НЕТ_В_УЧЁТЕ / ПОСТУПЛЕНИЕ_БЕЗ_ПАРЫ / РАСХОЖДЕНИЕ_ДАТ |

Раскрась столбец «Статус»:
- СВЕРЕНО → зелёная заливка
- РАСХОЖДЕНИЕ_ДАТ → жёлтая заливка
- НЕТ_В_УЧЁТЕ или ПОСТУПЛЕНИЕ_БЕЗ_ПАРЫ → красная заливка

### Лист 3: Задачи

Все открытые флаги из чек-листа. Столбцы:
| Столбец | Примечания |
|---|---|
| Категория | Операция без статьи / Нет первички / Дубль / Флаг сверки |
| Дата | Дата операции |
| Сумма | Сумма в рублях |
| Поставщик / Клиент | Наименование |
| Описание | Что не так и что делать |

Если открытых задач нет — покажи одну строку: «Открытых задач нет — учёт чист.»

## PDF-сводка — одна страница

Раскладка (сверху вниз):

```
[Название бизнеса]                  Пакет закрытия — [Месяц Год]
────────────────────────────────────────────────────────────

КЛЮЧЕВЫЕ ЦИФРЫ
Выручка        XX XXX ₽
Валовая маржа  XX%
Чистая прибыль XX XXX ₽

ОПиУ — СВОДКА
[Разбор простым языком на 150–250 слов из Шага 7]

ЗАДАЧИ
X операций без статьи · X нет первички · X открытых флагов
[или «Учёт чист — открытых задач нет.», если всё закрыто]

────────────────────────────────────────────────────────────
Подготовлено [Дата] · Собрано в Claude
```

Используй чистый шрифт без засечек (Helvetica, Inter или аналог). Логотип не
обязателен. Держи поля ≥ 2 см со всех сторон, чтобы печаталось аккуратно.

**Генерация PDF:** Используй библиотеку Python `xlsxwriter` или `reportlab`,
если запускаешь скрипт, либо отрисуй содержимое и экспортируй через печать в PDF
коннектором `~~хранилище` / Desktop. Собственник также может распечатать PDF из
листа ОПиУ в xlsx, если скрипт недоступен.
