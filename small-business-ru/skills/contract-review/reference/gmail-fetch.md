# Почта: поиск вложений-договоров

Используйте коннектор `~~почта` (например, Яндекс 360 или Mail.ru для бизнеса), чтобы найти вложения-договоры в недавних письмах.

> Имена методов ниже — пример интерфейса; адаптируйте под фактический коннектор. Если коннектора нет — используйте fallback.

## Поиск писем с договорами

Ищите недавние письма с вложениями по ключевым словам:

```
Запрос: есть вложение И (договор ИЛИ соглашение ИЛИ NDA ИЛИ контракт ИЛИ оферта) за последние 14 дней
```

## Что делать с результатами

1. Покажите короткий список (тема, отправитель, дата), если подходящих писем несколько.
2. Попросите владельца подтвердить, какое именно, перед скачиванием.
3. Получите письмо целиком, затем найдите и прочитайте вложение.

## Fallback

Если почта не подключена или подходящее вложение не найдено:

```
«Не нашёл недавнего вложения-договора в почте — перешлите письмо или приложите файл напрямую».
```

## Чего НЕ делать

- Не читайте письма, не относящиеся к договору (никакого сплошного перебора входящих).
- Не отправляйте ответы и не создавайте черновики писем во время проверки.
