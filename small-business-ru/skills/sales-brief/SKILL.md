Чуйка про «что хорошо идёт» врёт, а склад с зависшим товаром — нет. Скилл смотрит в цифры: что реально продалось (и что застряло), объясняет почему и собирает двухнедельный контент-план — продвигать лидеров, распродавать залежавшееся. Решения по данным, а не по ощущениям.

Разбор аргументов:
- `--lookback` (по умолчанию `30d`) — окно обзора `30d`, `60d` или `90d`

## Шаг 1 — Разбор продаж

По рабочему процессу анализа продаж:

1. Подтяните операции из `~~платежи` (ЮKassa, Тинькофф Касса) за окно обзора, сгруппированные по позиции/услуге/артикулу.
2. Подтяните выручку по категориям из `~~бухгалтерия` (1С:Бухгалтерия, МойСклад).
3. Ранжируйте товары по: общей выручке, объёму в штуках и марже (если есть в учёте).
4. Посчитайте долю каждого товара в выручке против прошлого эквивалентного периода.

Топ-продавцы: товары, нарастившие долю или удержавшие топ-3.
Слабые продавцы: товары с падающим объёмом или ниже 5% выручки.

## Шаг 2 — Проверка сезонности

1. Сравните текущий период с тем же периодом прошлого года (если есть история в учёте).
2. Пометьте позиции с сезонным паттерном (например, всплеск в 4 кв. (окт–дек), спад летом).
3. Отметьте новые товары без достаточной истории для определения сезонности.

## Шаг 3 — Разбор «почему»

Для каждого топового и слабого продавца объясните вероятную причину:
- изменение цены, акция, новый канал, сезонный спрос, ход конкурента
- сверьтесь с активностью кампаний в `~~crm` за период
- отметьте, где причина выведена логически, а где подтверждена

## Шаг 4 — Двухнедельный контент-бриф

Соберите готовый к работе контент-бриф:

```
Контент-бриф на 2 недели — {диапазон дат, ДД.ММ.ГГГГ}

ПРОДВИГАЕМ (лидеры)
• {товар}: {предлагаемый угол} — {канал: e-mail|соцсети|оба}
• {товар}: {предлагаемый угол} — {канал}

РАСПРОДАЁМ (залежавшееся)
• {товар}: {акция или идея комплекта} — {канал}

КОНТЕНТ-КАЛЕНДАРЬ
Неделя 1:
  Пн: {идея поста/письма}
  Ср: {идея поста/письма}
  Пт: {идея поста/письма}
Неделя 2:
  Пн: {идея поста/письма}
  Ср: {идея поста/письма}
  Пт: {идея поста/письма}
```

> Площадки РФ: Telegram, VK, Яндекс Дзен, Авито. Рассылки — Unisender, Sendsay. Помните про маркировку рекламы (ОРД, токен) для платных размещений.

### Пример (иллюстративный, данные условные)

Вход: выгрузка из ЮKassa за 30 дней, кофейня.

Выход (фрагмент брифа):

```
ПРОДВИГАЕМ (лидеры)
• Раф на кокосовом: +18% к прошлому месяцу, держит топ-3 — угол «сезонный
  фаворит, попробуй пока тепло» — соцсети
• Абонемент на 10 кофе: ровный спрос, высокая маржа — угол «считаем выгоду
  на чашке» — оба

РАСПРОДАЁМ (залежавшееся)
• Сезонный тыквенный сироп: ниже 5% выручки, сезон уходит — комплект
  «латте + сироп со скидкой» — соцсети
```

## Сбои коннекторов

Если недоступны и `~~бухгалтерия`, и `~~платежи` — остановитесь: разбор продаж требует хотя бы одного источника выручки. Если подключён один — работайте по нему и отметьте «бухгалтерия не подключена — данные о выручке только из платежей» (или наоборот). Если нет `~~crm` — пропустите сверку с кампаниями в «разборе почему» и отметьте.

## Точки подтверждения (approval gates)

- **Никогда не планируйте и не публикуйте контент автоматически.** Бриф только на проверку собственнику.
- **Не создавайте дизайн-макеты автоматически** — предложите собрать их через `~~дизайн` (VistaCreate, Supa) после одобрения брифа.

## Вывод

Покажите разбор продаж, затем контент-бриф. Спросите собственника, собрать ли макеты через `~~дизайн` для каких-то из запланированных постов.
