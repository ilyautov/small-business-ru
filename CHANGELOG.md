# Changelog

Версия пакета — единая в `small-business-ru/.claude-plugin/plugin.json` и
`.claude-plugin/marketplace.json` (metadata.version); при релизе бампаются вместе
и проставляется git-тег. Изменения отдельных killer-скиллов дополнительно ведутся
в их `SKILL.md` → `## Changelog`.

## [0.3.1-ru.1] — 2026-07-02

### Исправлено
- `smb-router`: маршрутизация к `counterparty-guard`, `tax-calendar-proactive`,
  `job-post-builder`; корзины «Контрагенты», «Найм» в обзоре; убрана ложная
  предпосылка «у всех скиллов есть команды».
- `tax_calc.py`: заработал `--vznosy-uplacheno` (новые поля «взносы_к_доплате»,
  «итого_к_доплате»); валидация отрицательных входов; код выхода 1 при ошибке;
  явное предупреждение при убытке на УСН-15 и при годах ≠ 2026.
- `fetch_counterparty.py`: проверка контрольного числа ИНН (алгоритм ФНС);
  явные блоки «фссп»/«суды»/«банкротство» со статусом «не собрано скриптом»,
  чтобы досье из трёх источников ФНС не выглядело полным; общий бюджет времени
  `COUNTERPARTY_DEADLINE` (по умолчанию 120 с — раньше при недоступной сети
  скрипт мог висеть 5+ минут); ретраи с backoff на HTTP 429/5xx.
- Версии синхронизированы: marketplace.json = plugin.json;
  metadata.version killer-скиллов подтянуты к их changelog.
- `.mcp.json` → `.mcp.example.json`: заготовки серверов с пустыми url больше
  не грузятся плагином (висели бы «failed» в /mcp) — теперь это документированный
  шаблон под self-hosted коннекторы.

### Добавлено
- eval: кейсы ошибок (отрицательный доход, невалидный режим/год), убыток УСН-15,
  исторические константы 2025, `--vznosy-uplacheno`; сверка встроенного зеркала
  канона 2026 в `tax_calc.py` с `tax_data.py`.
- Линтер: битые относительные ссылки в markdown; сверка README пака со списком
  скиллов.
- CI: защита от закоммиченных артефактов (.DS_Store, __pycache__, .pyc).
- Шаблоны issue (баг / спорная цифра / идея) и PR.
- Адаптеры: штамп синхронизации с версиями SKILL.md в каждом файле; таблица
  переносимости дополнена Gemini и Cursor.
- Сайт: favicon, canonical, og:url; навигация выровнена между страницами.
