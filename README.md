# small-business-ru ⚠️ alpha

**Открытый набор AI-скиллов для владельца малого бизнеса в РФ** — для Claude Code и Cowork (десктоп-приложение Claude). 34 готовых сценария: деньги, налоги УСН, дебиторка, маржа, продажи/CRM, контент, клиенты, найм — плюс killer-скиллы под российские реалии. Скажите обычными словами, что нужно («хватит ли на зарплату», «проверь контрагента», «закрой месяц»), и Claude подберёт нужный сценарий и проведёт по шагам с подтверждением на каждом шаге, который трогает деньги или клиентов.

> ⚠️ **alpha.** Помогает с рабочими задачами, но не является финансовой/налоговой/юридической/кадровой консультацией. Налоговые цифры — на 2026 год; сверяйте с действующими нормами и со своим бухгалтером/юристом.

## Установка

```text
/plugin marketplace add ilyautov/small-business-ru
/plugin install small-business-ru@small-business-ru
```

Затем скажите **«настрой меня»** — скилл `smb-onboard` поможет Claude понять ваш бизнес и подключить инструменты.

## Что внутри (34 скилла)

Покрыты основные зоны владельца-универсала: **деньги** (прогноз денежного потока, дебиторка, маржа, закрытие месяца), **налоги** (авансы УСН, взносы ИП, выплаты подрядчикам), **продажи/CRM** (приоритет лидов, гигиена CRM, кампании), **контент/маркетинг**, **клиенты** (жалобы, отзывы, поддержка), **найм** (вакансия/интервью/оффер), **брифы** (понедельник/пятница/квартал) и роутер `smb-router`, понимающий обычную речь.

**Killer-скиллы под РФ (наша оригинальная разработка):**
- **`counterparty-guard`** — проверка контрагента (юрлицо/ИП) **по ИНН** перед сделкой: ЕГРЮЛ, ФССП, банкротства, суды, светофор риска (расчёт кодом).
- **`tax-calendar-proactive`** — проактивный **налоговый штурман УСН**: по 3-5 фактам о бизнесе строит персональный календарь сроков и платежей.
- **`cross-source-verify`** — движок круговой сверки данных из нескольких источников.

Подключение РФ-сервисов (1С, ЮKassa, Битрикс24, Контур.Диадок, Яндекс 360, Telegram и др.) — опционально; без коннекторов скиллы работают через выгрузку CSV/Excel. Правила, налоговый/правовой слой и глоссарий — в [`small-business-ru/RULES.md`](./small-business-ru/RULES.md).
